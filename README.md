Churn Analysis project

Link to full project - https://docs.google.com/spreadsheets/d/16CKknTPssxpQiW1FhShqKyBaJ_8--g6Fgt4L2ocmKL4/edit?gid=1794670732#gid=1794670732

Data from Kaggle Telco Customer Churn dataset - https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Churn analysis is the process of evaluating customer data to understand why and when users cancel a product or service. In this project, I completed a churn analysis using roughly 7,000 records from the Telco Customer Churn dataset from Kaggle. I built a segmented metrics summary to calculate churn rates across contract type, tenure band, and internet service type, and visualized these findings with three corresponding charts. Lastly, I constructed a cohort-style retention heatmap using conditional formatting to depict churn patterns across customer segments.
